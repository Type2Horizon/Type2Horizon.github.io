How to Hack Hacker News Code

Hacker News is a popular website run by Y Combinator. It’s widely known by people in the tech industry as a community site for sharing news, showing off projects, asking questions, among other things.

In this project, you will be working with a table named hacker_news that contains stories from Hacker News since its launch in 2007. It has the following columns:

  • title: the title of the story
  • user: the user who submitted the story
  • score: the score of the story
  • timestamp: the time of the story
  • url: the link of the story

This data was kindly made publicly available under the MIT license.

  1. /* Returns the top five most popular Hacker News stories by score */
  2. SELECT title, score
  3. FROM hacker_news
  4. ORDER BY score DESC
  5. LIMIT 5;
  1. /* Returns the total score of all the stories */
  2. SELECT SUM(score)
  3. FROM hacker_news;
  1. /* Returns the individual users who have gotten combined scores of more than 200, and their combined scores */
  2. SELECT user, SUM(score)
  3. FROM hacker_news
  4. GROUP BY user
  5. HAVING SUM(score) > 200
  6. ORDER BY 2 DESC;
  1. /* Adds previous users’ scores together and divides by the total to get the percentage */
  2. SELECT (517 + 309 + 304 + 282) / 6366.0;
  1. /* Returns the number of users who posted this link: https://www.youtube.com/watch?v=dQw4w9WgXcQ */
  2. SELECT user,
  3.  COUNT(*)
  4. FROM hacker_news
  5. WHERE url LIKE '%watch?v=dQw4w9WgXcQ%'
  6. GROUP BY user
  7. ORDER BY COUNT(*) DESC;
  1. /* Returns which of these sites feed Hacker News the most: GitHub, Medium, or New York Times? */
  2. SELECT CASE
  3.   WHEN url LIKE '%github.com%' THEN 'GitHub'
  4.   WHEN url LIKE '%medium.com%' THEN 'Medium'
  5.   WHEN url LIKE '%nytimes.com%' THEN 'New York Times'
  6.   ELSE 'Other'
  7.  END AS 'Source',
  8.  COUNT(*)
  9. FROM hacker_news
  10. GROUP BY 1;
  1. /* Returns the best time of the day to post a story on Hacker News to get a high score */
  2. SELECT strftime('%H', timestamp) AS 'Hour',
  3.  ROUND(AVG(score), 1) AS 'Average Score',
  4. COUNT(*) AS 'Number of Stories'
  5. FROM hacker_news
  6. WHERE timestamp IS NOT NULL
  7. GROUP BY 1
  8. ORDER BY 1;
